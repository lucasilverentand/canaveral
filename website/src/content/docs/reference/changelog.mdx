# Changelog

## v0.1.0 (Upcoming)

Initial release with core functionality:

### Features

- **Framework Support**
  - Flutter (iOS, Android, macOS, Windows, Linux, Web)
  - Expo (iOS, Android)
  - React Native (iOS, Android)
  - Native iOS (Xcode)
  - Native Android (Gradle)
  - Tauri (macOS, Windows, Linux)

- **Distribution**
  - App Store Connect integration
  - Google Play Console integration
  - TestFlight automation
  - Firebase App Distribution

- **Code Signing**
  - Match-style certificate sync
  - Git, S3, GCS, Azure storage backends
  - Automatic certificate/profile creation

- **CI/CD**
  - GitHub Actions integration
  - GitLab CI integration
  - JSON output format
  - Structured error handling

### Commands

- `canaveral build` - Build apps
- `canaveral test` - Run tests
- `canaveral version` - Manage versions
- `canaveral upload` - Upload to stores
- `canaveral testflight` - TestFlight management
- `canaveral match` - Certificate sync
- `canaveral screenshots` - Screenshot automation
- `canaveral metadata` - Metadata management
- `canaveral doctor` - Environment check
