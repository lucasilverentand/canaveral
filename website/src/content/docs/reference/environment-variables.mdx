# Environment Variables

## iOS / App Store Connect

| Variable | Description |
|----------|-------------|
| `APP_STORE_CONNECT_API_KEY_ID` | App Store Connect API key ID |
| `APP_STORE_CONNECT_ISSUER_ID` | App Store Connect issuer ID |
| `APP_STORE_CONNECT_API_KEY` | API key content (PEM string) or path to .p8 file |
| `APP_STORE_CONNECT_API_KEY_PATH` | Path to .p8 key file |
| `APP_STORE_CONNECT_TEAM_ID` | Apple Developer Team ID (optional) |

## Certificate Sync (Match)

| Variable | Description |
|----------|-------------|
| `MATCH_GIT_URL` | Git URL for certificate storage |
| `MATCH_PASSWORD` | Certificate encryption password |

## Android / Google Play

| Variable | Description |
|----------|-------------|
| `GOOGLE_PLAY_SERVICE_ACCOUNT_KEY` | Service account JSON content or path to file |
| `GOOGLE_APPLICATION_CREDENTIALS` | Google Cloud credentials file path |
| `ANDROID_KEYSTORE_PATH` | Path to keystore file |
| `ANDROID_KEYSTORE_PASSWORD` | Keystore password |
| `ANDROID_KEY_ALIAS` | Signing key alias |
| `ANDROID_KEY_PASSWORD` | Signing key password |

## Firebase

| Variable | Description |
|----------|-------------|
| `FIREBASE_PROJECT_ID` | Firebase project ID |
| `FIREBASE_APP_ID` | Firebase app ID |
| `FIREBASE_TOKEN` | Firebase CLI token |
| `FIREBASE_SERVICE_ACCOUNT` | Firebase service account path |
| `GOOGLE_APPLICATION_CREDENTIALS` | Google Cloud credentials file path |

## Package Registries

| Variable | Description |
|----------|-------------|
| `NPM_TOKEN` | npm registry authentication token |
| `CARGO_REGISTRY_TOKEN` | crates.io API token |
