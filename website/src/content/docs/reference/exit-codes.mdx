# Exit Codes

| Code | Description |
|------|-------------|
| 0 | Success |
| 1 | General error / Command failed |
| 2 | Configuration error |
| 3 | Missing dependencies |
| 4 | Authentication error |
| 5 | Network error |
| 10 | Build failed |
| 11 | Test failed |
| 12 | Upload failed |
| 20 | Signing error |
