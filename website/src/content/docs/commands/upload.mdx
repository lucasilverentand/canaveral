# canaveral publish

Publish packages to registries and upload builds to app stores. Supports npm, crates.io, Apple App Store, Google Play, and Microsoft Store.

## Synopsis

```bash
canaveral publish <TARGET> [OPTIONS]
```

## Targets

| Target | Description |
|--------|-------------|
| `npm` | Publish to npm registry |
| `crates` | Publish to crates.io |
| `apple` | Submit to Apple App Store Connect |
| `google-play` | Upload to Google Play Console |
| `microsoft` | Submit to Microsoft Store |

## npm

```bash
# Publish package to npm
canaveral publish npm

# Publish with specific tag
canaveral publish npm --tag beta

# Dry run
canaveral publish npm --dry-run
```

## crates

```bash
# Publish to crates.io
canaveral publish crates

# Dry run
canaveral publish crates --dry-run
```

## apple

```bash
# Submit to App Store Connect
canaveral publish apple --artifact build/ios/MyApp.ipa
```

## google-play

```bash
# Upload to internal testing track
canaveral publish google-play --artifact build/android/app.aab --track internal

# Upload to production
canaveral publish google-play --artifact build/android/app.aab --track production
```

## microsoft

```bash
# Submit to Microsoft Store
canaveral publish microsoft --artifact build/windows/MyApp.msix
```

## Environment Variables

### npm

| Variable | Description |
|----------|-------------|
| `NPM_TOKEN` | npm registry authentication token |

### crates.io

| Variable | Description |
|----------|-------------|
| `CARGO_REGISTRY_TOKEN` | crates.io API token |

### Apple App Store Connect

| Variable | Description |
|----------|-------------|
| `APP_STORE_CONNECT_API_KEY_ID` | API key ID |
| `APP_STORE_CONNECT_ISSUER_ID` | API key issuer ID |
| `APP_STORE_CONNECT_API_KEY` | API key content (PEM) or path to .p8 file |
| `APP_STORE_CONNECT_API_KEY_PATH` | Path to .p8 key file |
| `APP_STORE_CONNECT_TEAM_ID` | Team ID (optional) |

### Google Play

| Variable | Description |
|----------|-------------|
| `GOOGLE_PLAY_SERVICE_ACCOUNT_KEY` | Service account JSON content or path to file |
| `GOOGLE_APPLICATION_CREDENTIALS` | Google Cloud credentials file path |
